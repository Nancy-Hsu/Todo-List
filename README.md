## 功能
- 可以註冊成會員，管理自己的 todo-list
- 也可以使用臉書登入
- 成為會員能夠 新增/編輯/刪除 todo-list
## 開發環境
- Node.js
- mysql2, sequelize, sequelize-cli
- express
- express-handlebars
- express-session
- passport, passport-facebook, passport-local
